## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/expense_app
```

Go to the project directory

```bash
  cd expense_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
